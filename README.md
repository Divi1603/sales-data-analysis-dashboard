# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview
This project analyzes real-world sales data to extract meaningful business insights such as monthly sales trends, top-performing cities, and customer purchasing behavior.  
An interactive dashboard is built using Streamlit for easy data visualization.

---

## 🚀 Features
- 📅 Monthly sales analysis  
- 🏙️ Sales distribution by city  
- ⏰ Peak order time identification  
- 🛒 Top-selling products  
- 📈 Interactive dashboard using Streamlit  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- Matplotlib  
- Streamlit
